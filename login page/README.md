## 🔧 Customization

### Colors
Update the CSS custom properties in `Lo Style.css`:
```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --success-color: #27ae60;
  --error-color: #e74c3c;
}
```

### Animations
Modify animation durations and effects in the CSS file:
```css
.transition {
  transition: all 0.3s ease;
}
```
